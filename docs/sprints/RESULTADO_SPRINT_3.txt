📞 GUIA RÁPIDO - USO DOS ENDPOINTS

  1. LOGIN (gerar tokens):
     POST /sistema/autenticar.php
     Body: email, senha
     → Tokens salvos automaticamente no localStorage

  2. RENOVAR TOKEN (quando expirar):
     POST /apiIgreja/auth/token.php
     Body: { "refresh_token": "uuid-v4" }
     → Response: { access_token, refresh_token, expires_in, user }

  3. LOGOUT (revogar token):
     POST /apiIgreja/auth/logout.php
     Body: { "refresh_token": "uuid-v4" }
     → Response: { success: true, message: "Logout realizado" }

  4. REQUISIÇÕES AUTENTICADAS:
     Header: Authorization: Bearer {access_token}
